When you click 'submit', a text file with the URLs of the files will get downloaded. The URLs are valid for one week.

To download the files:

  • With Firefox and DownThemAll: Click Tools - DownThemAll! - Manager. Right click - Advanced - Import from file. Right-click - Select All. Right-click - Toogle All
  • With Chrome and TabToSave: Click the T/S icon next to the URL bar, click the edit button at the bottom of the screen and paste the file contents
  • OSX/Linux: With curl and a single thread: xargs -n1 curl -JO < fileUrls.txt
  • Linux: With wget and a single thread: wget --content-disposition -i fileUrls.txt
  • With wget and 4 threads: xargs -n 1 -P 4 wget --content-disposition -q < fileUrls.txt
  • With aria2c, 16 threads and two threads per file: aria2c -x 16 -s 2 -i fileUrls.txt
When you click 'submit', a shell script that runs curl will get downloaded. The URLs are valid for one week.

Then, to download the files:

  • Linux/OSX: With curl and a single thread: sh downloadCirm.sh
  • Linux/OSX: With curl and four threads: parallel -j4 :::: downloadCirm.sh
